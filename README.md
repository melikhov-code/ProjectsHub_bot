# Projects Hub Telegram Bot  
## Requirements  
1. python  
2. telebot  
3. sqlite3  

## Setup  
### Token & Run  
1 Step: In the Telegram, search for @BotFather.  
2 Step: Create a new bot and copy it's token.  
3 Step: Go to the file config.py, and paste your token into similar variable.  
4 Step: Run every script following this order:  
  1. config.py  
  2. logic.py  
  3. main.py
